## Automação de Testes - Playwright

Este portfólio também possui um projeto inicial de automação de testes utilizando Playwright e JavaScript.

A automação foi criada para validar cenários de login no BugBank, complementando os testes manuais já documentados no projeto.

### Cenários automatizados

- Login com campos vazios
- Login com senha inválida
- Login com sucesso
