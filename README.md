Este repositório faz parte do meu portfólio de estudos em QA/Testes de Software.
O objetivo do projeto é documentar testes manuais para um sistema fictício de chamados, incluindo plano de testes, casos de teste, relatório de bugs, evidências e estudos iniciais em automação de testes.

## Objetivos
- Praticar a criação de casos de teste
- Documentar cenários positivos e negativos
- Registrar bugs de forma clara e organizada
- Trabalhar com evidências de testes
- Evoluir meus estudos em QA Manual e Automação de Testes
