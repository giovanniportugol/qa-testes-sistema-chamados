# QA Portfolio — Sistema de Chamados, BugBank e Automação Playwright

Projeto acadêmico em desenvolvimento para estudo, prática e transição de carreira para a área de Tecnologia da Informação, com foco inicial em QA/Testes de Software.

---

## Sobre o projeto

Este repositório foi criado para registrar minha evolução nos estudos iniciais em **QA/Testes de Software**, como parte da minha transição para a área de **Tecnologia da Informação**.

Atualmente curso **Análise e Desenvolvimento de Sistemas pela Estácio** e estou desenvolvendo conhecimentos acadêmicos em testes manuais, testes funcionais, documentação de bugs, evidências, matriz de rastreabilidade e automação inicial com **Playwright** e **JavaScript**.

O foco do projeto é aprender, de forma prática e organizada, como estruturar um portfólio de QA, utilizando um sistema de chamados, estudos com BugBank e primeiros exercícios de automação.

---

## Objetivo do projeto

Demonstrar minha evolução nos estudos de QA, praticando conceitos como:

## Escopo de estudo

O projeto utiliza como base um sistema de chamados e estudos com o **BugBank**, com foco em cenários simples de aprendizado.

Os estudos incluem:

- Validação de fluxo de login
- Escrita de casos de teste
- Registro e documentação de bugs
- Organização de evidências
- Criação de documentação básica de QA
- Primeiros estudos de automação com Playwright
- Prática de JavaScript aplicada a testes
