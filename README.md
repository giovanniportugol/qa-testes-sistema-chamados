# QA Portfolio — Sistema de Chamados, BugBank e Automação Playwright

Projeto acadêmico em desenvolvimento para estudo de **QA/Testes de Software**.

---

## Sobre o projeto

Este repositório foi criado para registrar minha evolução nos estudos iniciais em **QA/Testes de Software**.

O foco do projeto é aprender, de forma prática e organizada, conceitos fundamentais de qualidade de software, testes manuais, documentação de QA, evidências e primeiros estudos em automação.

Este projeto faz parte da minha transição para a área de **Tecnologia da Informação**.

---

## Objetivo

Demonstrar minha evolução nos estudos de QA, aprendendo conceitos como:

- O que é QA
- Testes manuais
- Testes funcionais
- Plano de testes
- Casos de teste
- Relatório de bugs
- Evidências de testes
- Matriz de rastreabilidade
- Automação inicial com Playwright
- Organização de documentação técnica

---

## Escopo de estudo

O projeto utiliza como base um sistema fictício de chamados e estudos com o **BugBank**, com foco em cenários simples de aprendizado.

Os estudos incluem:

- Validação de fluxo de login
- Registro de bugs encontrados
- Organização de evidências
- Escrita de casos de teste
- Criação de documentação básica
- Primeiros testes automatizados com Playwright
