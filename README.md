# 🧪 Portfólio de Controle de Qualidade — Sistema de Chamados

Projeto de portfólio desenvolvido para demonstrar conhecimentos em Quality Assurance (QA), incluindo documentação de testes, testes manuais, testes de APIs REST, automação com Playwright e boas práticas de qualidade de software.

---

# 🎯 Objetivo do Projeto

Documentar e organizar testes para um sistema fictício de chamados, simulando etapas utilizadas em projetos reais de QA em um ambiente de estudo.

O projeto contempla:

* Plano de Testes
* Casos de Teste
* Relatórios de Bugs
* Matriz de Rastreabilidade
* Evidências de Testes
* Testes de APIs REST
* Automação de Testes com Playwright
* Projeto Prático utilizando o BugBank

---
