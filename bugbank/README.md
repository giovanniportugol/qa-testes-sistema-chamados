# Projeto BugBank — Testes Manuais

## Objetivo

Esta pasta contém a documentação de testes manuais realizados no site BugBank,  
utilizado como ambiente de prática para estudos em QA/Testes de Software.

O objetivo é simular cenários reais de validação de funcionalidades,  
registrando casos de teste, bugs encontrados e evidências de execução.

---

## Sobre o BugBank

O BugBank é uma aplicação web utilizada para prática de testes,  
simulando funcionalidades de um banco digital.

Neste projeto, o BugBank foi utilizado com finalidade acadêmica e de portfólio,  
para praticar análise de funcionalidades, criação de cenários de teste,  
execução manual e documentação de resultados.

---

## Funcionalidades Testadas

Foram considerados cenários relacionados às seguintes funcionalidades:

- Cadastro de usuário
- Login
- Validação de campos obrigatórios
- Transferência entre contas
- Consulta de extrato
- Mensagens de erro
- Cenários positivos e negativos
