# Projeto de Testes - BugBank

Este diretório contém a documentação dos testes manuais realizados no site BugBank, utilizado como ambiente de prática para estudos em QA/Testes de Software.

O objetivo é aplicar conceitos de qualidade de software em um sistema web real de treinamento, criando plano de testes, casos de teste, relatório de bugs e evidências.

## Site testado

BugBank  
https://bugbank.netlify.app/

## Objetivos

- Praticar testes manuais em uma aplicação web
- Criar cenários de teste positivos e negativos
- Documentar casos de teste
- Registrar bugs encontrados
- Organizar evidências de execução dos testes
- Fortalecer meu portfólio de QA no GitHub

## Funcionalidades a serem testadas

- Cadastro de usuário
- Login
- Validação de campos obrigatórios
- Transferência entre contas
- Consulta de extrato
- Mensagens de erro e sucesso
